# 1.4 Deploy to Devnet

## Mục tiêu
Build và deploy Solana Program lên Devnet, lấy Program ID.

## Steps

### 1. Config Anchor.toml
```toml
[programs.devnet]
deadmans_switch = "<PROGRAM_ID>"

[provider]
cluster = "devnet"
wallet = "~/.config/solana/id.json"
```

### 2. Build
```bash
cd anchor
anchor build
```

### 3. Get Program ID
```bash
solana address -k target/deploy/deadmans_switch-keypair.json
```
→ Copy ID này vào `Anchor.toml` và `lib.rs`

### 4. Deploy
```bash
anchor deploy --provider.cluster devnet
```

### 5. Verify
```bash
solana program show <PROGRAM_ID>
```

## Frontend Config
Tạo file `src/utils/anchor.ts`:
```typescript
export const PROGRAM_ID = new PublicKey("<PROGRAM_ID>");
export const connection = new Connection(clusterApiUrl('devnet'));
```

## Troubleshooting
- **Insufficient funds:** `solana airdrop 2 --url devnet`
- **Program too large:** Optimize account sizes
