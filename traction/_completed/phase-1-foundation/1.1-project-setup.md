# 1.1 Project Setup

## Mục tiêu
Khởi tạo monorepo với Next.js (frontend) và Anchor (smart contract).

## Directory Structure
```
deadmans-switch/
├── anchor/                 # Solana Program
│   ├── programs/deadmans-switch/src/
│   ├── tests/
│   └── Anchor.toml
├── src/                    # Next.js App
│   ├── app/
│   ├── components/
│   └── utils/
├── package.json
└── next.config.js
```

## Tasks

### 1. Init Next.js
- Dùng `npx create-next-app@latest` với App Router
- TypeScript enabled
- TailwindCSS enabled

### 2. Init Anchor
- Chạy `anchor init anchor` trong project root
- Hoặc tạo folder `anchor/` riêng với `anchor new deadmans-switch`

### 3. Install Dependencies
**Frontend:**
- `@solana/web3.js`
- `@coral-xyz/anchor`
- `@solana/wallet-adapter-react`
- `@solana/wallet-adapter-wallets`

**Dev:**
- Anchor CLI (`cargo install --git https://github.com/coral-xyz/anchor avm`)

## Acceptance Criteria
- [ ] `npm run dev` chạy Next.js ở localhost:3000
- [ ] `anchor build` thành công trong folder `anchor/`
- [ ] Không có lỗi TypeScript
