# 1.3 Wallet Connect

## Mục tiêu
Tích hợp Solana Wallet Adapter để connect Phantom/Solflare.

## Components cần tạo

### 1. `WalletContextProvider.tsx`
Wrap app với các providers:
- `ConnectionProvider` - RPC endpoint
- `WalletProvider` - supported wallets
- `WalletModalProvider` - UI modal

### 2. `WalletButton.tsx`
Button hiển thị:
- "Connect Wallet" khi chưa connect
- Địa chỉ rút gọn + Disconnect khi đã connect

## Config

### RPC Endpoints
```typescript
const endpoint = clusterApiUrl('devnet'); // MVP
// Mainnet: dùng private RPC (Helius, QuickNode)
```

### Supported Wallets
```typescript
const wallets = [
  new PhantomWalletAdapter(),
  new SolflareWalletAdapter(),
];
```

## UI Requirements
- Button style theo design system (dark theme)
- Modal wallet selection đẹp
- Hiển thị balance SOL (optional cho MVP)

## Acceptance Criteria
- [ ] Click "Connect" → hiện modal chọn wallet
- [ ] Connect Phantom thành công
- [ ] Địa chỉ hiển thị dạng `AbCd...xYz`
- [ ] Có thể Disconnect
