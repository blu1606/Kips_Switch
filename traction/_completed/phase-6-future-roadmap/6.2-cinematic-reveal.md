## üéØ Goal
Transform the claim experience from a transactional file download into an emotional, respectful ceremony.

## üé® Visual Concept

### State 1: Locked
- **Visual:** A high-quality 3D Safe or Cyberpunk Monolith (using `Spline` or purely CSS 3D).
- **Animation:** Breathing glow, subtle rotation.
- **Interaction:** Mouse move affects lighting/perspective.

### State 2: The Reveal (Transition)
- **Trigger:** Decryption successful.
- **Animation:**
    1.  Main interface fades to black.
    2.  Sound design: Heavy mechanical unlocking sound + ethereal "release" chime.
    3.  The Safe/Monolith cracks open or dissolves into particles.

### State 3: The Message
- **Content:** If the owner left a text note or video, it plays **first**, alone on screen.
- **Context:** "A final message from [Name]"
- **Pacing:** User must finish reading/watching (or click "Continue") before seeing assets.

### State 4: Inheritance
- **Visual:** Grid of artifacts (Files, Tokens, Keys) fades in.
- **Style:** Premium cards, glassmorphism.
