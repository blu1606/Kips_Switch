# 15.1: Passkey-Native Onboarding

## Overview
Enable users to create KipSwitch vaults and manage assets using biometric authentication (FaceID, TouchID, Windows Hello) via LazorKit.

## User Flow
1. **Selection:** User chooses "Connect with Biometrics" on the landing page.
2. **Creation:** Browser/OS prompts user to create a Passkey.
3. **Smart Wallet Setup:** LazorKit creates a PDA-based smart account controlled by the Passkey.
4. **Seamless Access:** On subsequent visits, user logs in with a quick biometric check.

## Technical Details
- **Dependency:** `@lazorkit/wallet-sdk`
- **Component:** `LazorkitProvider` bọc ngoài `App` hoặc `Providers`.
- **Primary Auth:** WebAuthn standard via LazorKit on-chain program.
- **Account Type:** Program Derived Address (PDA).
